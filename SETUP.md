# Set-Up For Heart Disease Detector Project

- To install all the required dependencies for this project, follow the following process (steps 1-3 should already have been completed after following the "Quick Start" section of README.md, but are repeated below just in case):

**1) Navigate to the GitHub repository**
- URL: https://github.com/edominicduke/heart-disease-detector.git

**2) Clone the GitHub repository**
- Command: git clone https://github.com/edominicduke/heart-disease-detector.git

**3) Change into the appropriate directory**
- Command: cd heart-disease-detector

**4) Install all the dependencies listed in requirements.txt**
- Command: pip install -r requirements.txt
